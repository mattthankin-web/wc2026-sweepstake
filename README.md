# WC2026 Sweepstake — Live Site

A mobile-first live sweepstake tracker for 11 participants following the 2026 FIFA World Cup.

- **Scores** refresh every 30 minutes automatically
- **Commentary** generated by Claude AI daily at 2pm AEST
- **Hosted free** on GitHub Pages — shareable URL, works on any phone browser

---

## Deploy in 10 minutes

### 1. Create a GitHub account
Go to [github.com](https://github.com) and sign up (free).

### 2. Create a new repository
- Click **+** → **New repository**
- Name it: `wc2026-sweepstake`
- Set to **Public** (required for free GitHub Pages)
- Do **not** initialise with a README
- Click **Create repository**

### 3. Upload the files
- On the new repo page, click **uploading an existing file**
- Upload everything from this folder, maintaining the structure:
  ```
  index.html
  data/
    data.json
  scripts/
    fetch_scores.py
    generate_commentary.py
  .github/
    workflows/
      fetch-scores.yml
      generate-commentary.yml
  ```
- Commit with message: `initial commit`

### 4. Enable GitHub Pages
- Go to repo **Settings** → **Pages** (left sidebar)
- Under **Source**, select **Deploy from a branch**
- Branch: `main`, folder: `/ (root)`
- Click **Save**
- Your site URL will appear: `https://YOUR-USERNAME.github.io/wc2026-sweepstake`
- Takes ~2 minutes to go live

### 5. Add API keys as secrets
- Go to repo **Settings** → **Secrets and variables** → **Actions**
- Click **New repository secret** for each:

| Secret name | Value | Where to get it |
|---|---|---|
| `FOOTBALL_API_KEY` | Your API key | [football-data.org](https://www.football-data.org/client/register) — free account, instant |
| `ANTHROPIC_API_KEY` | Your Claude API key | [console.anthropic.com](https://console.anthropic.com) → API Keys |

### 6. Test the workflows
- Go to **Actions** tab in your repo
- Click **Fetch Scores** → **Run workflow** → **Run workflow**
- Watch it run (takes ~30 seconds)
- Check that `data/data.json` updated in your repo

### 7. Share the URL
```
https://YOUR-USERNAME.github.io/wc2026-sweepstake
```
Send that to all 11 participants. Works on any phone browser, no app needed.

## How it works

```
football-data.org API
        ↓ (every 30 min via GitHub Actions)
scripts/fetch_scores.py
        ↓
data/data.json  ←──────────── Claude API (daily 2pm AEST)
        ↓                            ↑
index.html                  scripts/generate_commentary.py
(reads data.json on load,
 auto-refreshes every 5 min)
```

---

## Updating the site

**Win percentages** — `fetch_scores.py` calculates on-pitch stats automatically. Win% (Kalshi odds) needs updating manually in `data/data.json` when odds shift significantly. Edit the `win_pct` field in the `standings` array and commit.

**Odds table** — Edit the `odds` array in `data/data.json` and commit.

**Upcoming fixtures context notes** — Edit the `context` field in `upcoming_fixtures` in `data/data.json` and commit.

**Standing instructions for commentary** — Edit `PARTICIPANT_NOTES` in `scripts/generate_commentary.py`. The Galbraith/Melbourne Demons angle is already in there.

**Force a commentary regeneration** — Go to Actions → Generate Commentary → Run workflow.
